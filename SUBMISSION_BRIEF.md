# MoonGridGym Submission Brief

MoonGridGym is a lightweight Gym-style environment suite for MoonBit.

## Links

- GitLink: https://gitlink.org.cn/zyjzyj78/MoonGridGym
- GitHub: https://github.com/zhaoyuanjuns/MoonGridGym

## Highlights

- Unified `reset / step / render` API
- Classic grid environments with seeded variants
- Built-in BFS shortest-path planning
- Auto-solve and rollout helpers for demos
- Tests and CI included

## Compliance posture

- Single contributor history
- MIT licensed
- Public repository layout
- Original implementation
- Contest-friendly scope and documentation
